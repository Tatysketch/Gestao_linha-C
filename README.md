# 📊 Sistema de Gestão Tática - Linha C (P2M)

### 🚀 Visão Geral
Este projeto é uma ferramenta de **Business Intelligence (BI) Operacional** desenvolvida para otimizar o controle de fluxo, presença e performance técnica dos colaboradores na Linha C. O objetivo é substituir controles manuais por um dashboard digital de alta visibilidade.

## 🛠 Funcionalidades Chave
*   **Check-in de Presença:** Controle em tempo real do absenteísmo.
*   **Mapa de Calor Técnico (Heatmap):** Identificação visual do nível de proficiência da equipe (N1 a N4).
*   **Cálculo de Capacidade Operacional:** Algoritmo que calcula a saúde da linha com base no Score Médio.
*   **Exportação de Dados:** Geração de relatórios em CSV para análise de tendências no Excel.
*   **Persistência Local:** Os dados são salvos no navegador (LocalStorage), garantindo que nenhuma informação seja perdida ao fechar a aba.

## 📈 Impacto na Operação
O sistema permite ao TL (Team Leader):
1. Identificar gargalos de treinamento instantaneamente.
2. Alocar recursos (como o Samuel na bancada de pesca) de forma estratégica.
3. Tomar decisões baseadas em dados (Data-Driven) para evitar quedas de produtividade.

## 💻 Tecnologias Utilizadas
*   **HTML5 / CSS3:** Interface responsiva com foco em conforto visual (Âmbar/Dark Mode).
*   **JavaScript (Vanilla):** Lógica de cálculo de métricas e manipulação de DOM.
*   **LocalStorage API:** Armazenamento de dados sem necessidade de banco de dados externo.
